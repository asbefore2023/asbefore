## ❌ אם זה לא עובד - פתרון בעיות

### 1. פתח Console בדפדפן
```
לחץ F12 → טאב "Console"
```

### 2. חפש שגיאות אדומות

#### אם רואה: "The public key is required"
```
❌ בעיה: המפתח לא נטען
✅ פתרון: רענן את הדפדפן (Ctrl+Shift+R)
```

#### אם רואה: "Service doesn't exist"
```
❌ בעיה: ה-Service ID לא נכון
✅ פתרון:
1. עבור ל-EmailJS Dashboard
2. Email Services → לחץ על השירות שלך
3. וודא שה-Service ID הוא: service_pnzfxna
```

#### אם רואה: "Template not found"
```
❌ בעיה: ה-Template לא נמצא או לא פעיל
✅ פתרון:
1. עבור ל-EmailJS Dashboard
2. Email Templates → לחץ על contact_form_hebrew
3. וודא שהתבנית פעילה (לא מושבתת)
4. וודא שה-Template ID הוא: template_1m4w18c
```
